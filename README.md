# ParisSortie

Une application web moderne pour découvrir les événements à Paris avec une interface minimaliste.

## Caractéristiques

- Interface utilisateur simple
- Système de pagination dynamique
- Affichage des événements avec images et détails
- Filtrage des événements
- Design responsive
- Thème sombre élégant

## Technologies utilisées

- HTML5
- CSS3 (Variables CSS personnalisées)
- JavaScript (Vanilla)
- Bootstrap 5
- API Paris Event (à spécifier)

## Installation

1. Clonez ce dépôt
```bash
git clone https://github.com/ettorhake/ParisSortie.git
```

2. Ouvrez le dossier du projet
```bash
cd ParisSortie
```

3. Ouvrez le fichier `index.html` dans votre navigateur ou utilisez un serveur local
